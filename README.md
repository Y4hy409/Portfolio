## About

This portfolio presents my work as a Computer Science student and frontend
developer. It includes my technical skills, selected projects, academic
background, certifications, and contact information in a clean, accessible
single-page experience.

## Highlights

- Responsive single-page layout
- Sticky navigation and smooth scrolling
- Technology and project showcases
- Certificate previews with verification links
- Accessible keyboard focus states
- No framework or build-tool dependency

## Built with

- Semantic HTML5
- Modern CSS with custom properties, Flexbox, and Grid
- Boxicons
- Google Fonts: Playfair Display and Inter

## Run locally

Clone the repository and open `index.html` in a browser:

```bash
git clone https://github.com/Y4hy409/Portfolio.git
cd Portfolio
```

For a local web server:

```bash
python -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).
